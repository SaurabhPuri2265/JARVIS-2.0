
# Tkinter

- The Library is used to implement GUI on python 3.7.x

- **USES**
  - Create and destroy windows.
  - Create buttons
  - Insert label widget
  - Insert text widget
  - and Many More----
  
  
- **Documentation**
  - Available at [effbot.org](https://effbot.org/tkinterbook/tkinter-index.htm#class-reference)
  

- **Installation**

```
pip install tk-tools
```
